# 한국어 Voice API 비교 분석

숏폼 영상 제작 시 TTS(텍스트 음성 변환) 연동을 위한 한국어 Voice API 3종 비교.

---

## 1. 네이버 클로바 보이스 (NAVER CLOVA Voice)

| 항목 | 내용 |
|------|------|
| **제공** | 네이버 클라우드 플랫폼 |
| **한국어 품질** | ⭐⭐⭐⭐⭐ (국내 최고 수준) |
| **가격** | Premium: 월 100만자 포함, 초과 시 1,000자 단위 과금 |
| **API** | REST API, 1회 최대 2,000자 |
| **파라미터** | Volume, Speed, Pitch, Emotion |
| **언어** | 한국어, 영어, 일본어, 중국어, 스페인어 |

**장점**
- 한국어 자연스러움 우수
- 국내 서비스, 문서·지원 한글
- 감정 파라미터 제공

**단점**
- 네이버 클라우드 가입 필요
- 초과 요금 구조 확인 필요

**추천 용도**: 한국어 숏폼 내레이션, 상업용

---

## 2. Google Cloud Text-to-Speech

| 항목 | 내용 |
|------|------|
| **제공** | Google Cloud Platform |
| **한국어 품질** | ⭐⭐⭐⭐ (WaveNet/Neural 고품질) |
| **가격** | 무료: 월 400만자 / Standard $4/100만자 / WaveNet $16/100만자 |
| **API** | REST, gRPC |
| **파라미터** | SSML 지원, 속도·피치·볼륨 |
| **언어** | 90개 이상, 400+ 음성 |

**장점**
- 무료 티어 넉넉함
- SSML로 자동화 용이
- 글로벌 안정성

**단점**
- 한국어 전용 음성 수는 제한적
- GCP 설정 필요

**추천 용도**: 대규모 처리, 다국어, 비용 효율

---

## 3. Microsoft Azure Speech (TTS)

| 항목 | 내용 |
|------|------|
| **제공** | Microsoft Azure |
| **한국어 품질** | ⭐⭐⭐⭐ (Neural 음성) |
| **가격** | 무료: 월 50만자 / Neural $16/100만자 |
| **API** | REST, SDK |
| **파라미터** | 스타일, 말투, 억양, 속도 |
| **언어** | 500종 이상 음성 |

**장점**
- 무료 50만자로 실무 테스트 가능
- 음성 스타일 세밀 제어
- 상업 이용 명확

**단점**
- Azure 계정 필요
- 한국어 Neural 음성 종류 확인 필요

**추천 용도**: 챗봇, 자동화, 상업 서비스
