## 수동 테스트 체크리스트

### 문제 1: 진행률/퍼센트
- [ ] 숏폼 제작 버튼 클릭 시 퍼센트 표시 없음
- [ ] 3초 폴링으로 상태만 갱신

### 문제 4: 검색어
- [ ] AI 추출 검색어: 삭제 버튼 없음, ⊙/○로 활성화·비활성화
- [ ] 사용자 추가 검색어: 삭제 가능, 20자 제한
- [ ] 21자 입력 시 저장 거부

### 문제 5: 멀티미디어
- [ ] URL 탭: URL 입력 후 "URL로 추가"
- [ ] 파일 탭: 파일 선택 후 "파일 업로드"
- [ ] 업로드된 이미지/영상 미리보기

### 문제 3: 한글 자막 (tofu)
- [ ] **Worker 재빌드 필수**: `docker compose build worker` 후 `docker compose up -d`
- [ ] 로컬 Worker: `.\worker\scripts\download-font.ps1` 실행 후 celery/consumer 기동
- [ ] 새로 렌더한 영상에서 한글 자막 표시 확인

### 문제 2: 렌더 실패
- [ ] Worker 중지 후 렌더 요청 → 실패 처리 확인
- [ ] 실패 시 에러 메시지 표시
